# 😀 Emotion Detection Dashboard | By Nisha Samreen

An AI-powered **Facial Emotion Detection Dashboard** developed by **Nisha Samreen** using **HTML, CSS, JavaScript, and Face API.js**.  
This project performs real-time facial emotion analysis through webcam input and dynamically displays detected emotions with confidence scores.

---

## 🚀 Features

- 🎥 Live webcam emotion detection
- 😀 Real-time facial expression analysis
- 📊 Interactive emotion dashboard
- ⚡ Fast browser-based processing
- 🖥️ Responsive and clean UI

### Supported Emotions
- 😊 Happy
- 😢 Sad
- 😡 Angry
- 😲 Surprised
- 😐 Neutral

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- Face API.js

## ▶️ How This Project Works

1. Webcam access is enabled through the browser  
2. Face API.js detects faces in real time  
3. Facial expressions are analyzed instantly  
4. Emotion confidence levels are displayed dynamically  
5. Current mood updates automatically on the dashboard
